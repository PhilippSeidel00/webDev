ROSE - Road System Editor

Rose (Road system editor) in a graphical editor used to create and manage road networks. These networks can then be exported into numerous formats to subsequently be used in other software (such as simulation tools). This group project was developed as part of the "Praxis der Softwareentwicklung"-course at KIT. Both documentation and the source code are publicly available.
